# Vibecoding Test — Dandy Solutions

## Project Structure

```
Vibecoding-Test/
├── index.html        # Main landing page
├── CLAUDE.md         # Project guide for Claude Code
├── assets/
│   ├── images/       # Photos, logos, hero images
│   └── icons/        # SVG or PNG icons
├── css/              # External stylesheets
├── js/               # JavaScript files
└── pages/            # Additional HTML pages (e.g. about.html, services.html)
```

## Stack

- Plain HTML, CSS, JavaScript (no framework)
- Google Fonts: Playfair Display + Inter
- Color scheme: light beige (#f5f0e8) and white (#ffffff), gold accent (#b8975a)

## Running Locally

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080
